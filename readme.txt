A extension to send SMS notification to vendors. SMS notification can be sent with customized contents.

== Description ==

Try for FREE. 20 trial SMS credits will be given upon [registration](https://dashboard.moceanapi.com/register?fr=wp). Additional SMS credits can be requested and is subject to approval by MoceanSMS.

Features:

* Notify vendor when there's new order

== Installation ==

1. This plugin required
- [MoceanSMS Order SMS Notification for WooCommerce](https://wordpress.org/plugins/moceansms-order-sms-notification-for-woocommerce/) >= Ver 1.0.8
- [Woocommerce Product Vendors](https://woocommerce.com/products/product-vendors/)

2. Search for "MoceanSMS WC Marketplace Extension" in "Add Plugin" page and activate it.
